# Machine Learning Kaggle Assignments

This repository contains solutions to several Kaggle-style machine learning competitions completed during coursework and self-study.  
The projects cover a wide range of machine learning tasks including classification, regression, sequence modeling, and scientific ML applications.


## Projects Overview

### Digit Recognizer (3 vs 8 Classification)
Binary image classification task based on distorted MNIST-style digit images.

- Task type: Computer Vision / Classification
- Goal: Classify handwritten digits 3 and 8
- Dataset: Transformed MNIST-like images
- Competition:
  https://www.kaggle.com/competitions/25-hse-recognizer/overview

Folder: `Recognizer`

---

### Phoneme Classification
Speech signal classification using acoustic spectral features.

- Task type: Tabular Classification / Speech ML
- Goal: Predict phoneme class from spectral features
- Dataset: TIMIT speech corpus derived dataset
- Competition:
  https://www.kaggle.com/competitions/hse-phoneme-5

Folder: `Phonemes`

---

### Vehicle MPG and Price Prediction
Multi-target regression task predicting automobile efficiency and price.

- Task type: Multi-output Regression
- Targets:
  - City MPG
  - Highway MPG
  - Vehicle price
- Dataset source: OpenML Automobile dataset
- Competition:
  https://www.kaggle.com/competitions/25-hse-auto/overview

Folder: `Auto`


### Bacteria DNA Sequence Classification
Genomic sequence classification problem.

- Task type: Sequence Classification / Bioinformatics ML
- Goal: Classify bacteria based on DNA sequences
- Dataset:
  https://www.tensorflow.org/datasets/catalog/genomics_ood
- Competition:
  https://www.kaggle.com/competitions/25-hse-genomics/overview

Folder: `Genomics`


### Stellar Object Classification
Astronomical object classification based on spectral features.

- Task type: Multiclass Classification
- Classes:
  - Star
  - Galaxy
  - Quasar
- Dataset: Sloan Digital Sky Survey (SDSS)
- Competition:
  https://www.kaggle.com/competitions/25-hse-stellar/overview

Folder: `Stellar`


### Particle Entry Position Reconstruction
Scientific regression task using particle detector data.

- Task type: Multi-output Regression
- Goal: Predict particle entry coordinates (x, y)
- Dataset: Electromagnetic calorimeter simulation
- Competition:
  https://www.kaggle.com/competitions/ml-1-hackathon/overview

Folder: `Hakathon`
